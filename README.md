# ODA-IDE

This is a simply IDE for C and C++ development.


## Compile

For compilation you need to have Qt5 and qmake5.

```bash
cd /desired/path/
git clone https://github.com/bartfaimate/ODA-IDE
mkdir build
cd build
qmake ../ODA-IDE.pro
make
```
